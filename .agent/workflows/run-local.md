To start the application locally, run the following command in the project root:

// turbo
1. Execute the local run script:
```bash
./scripts/run_local.sh
```

This will:
- Start the FastAPI backend on http://localhost:8000
- Start the Next.js frontend on http://localhost:3000

Make sure you have your `.env` file configured with `OPENROUTER_API_KEY`.
